# interimAttestationJava
Java-project

Необходимо написать проект, для розыгрыша в магазине игрушек.
Функционал должен содержать добавление новых игрушек и задания веса
для выпадения игрушек.

Задание
1) Используйте команды mkdir и cat операционной системы Linux и создайте две
директории:
- "Игрушки для школьников", заполнив файлами
"роботы", "конструктор", "настольные игры";
- "Игрушки для дошколят", заполнив файлами "мягкие игрушки", "куклы", "машинки",
2) Объединить две директории из предыдущего задания в одну с названием "Имя игрушки". Просмотреть
содержимое созданного каталога.
3) Переименовать директорию, присвоив ей новое имя "Игрушки" (повтор п. 2)
4) Установить и удалить snap-пакет.
5) Добавить задачу для выполнения каждые 3 минуты (запись во все файлы).

Java:
6) Создать файл "Призы" минимум из 3 строк, содержащих:
- id игрушки,
- текстовое название,
- частоту выпадение игрушки(веса в %)
7) Напишите класс, конструктор у которого принимает минимум 3 строки, содержащие три поля:
- id игрушки,
- текстовое название,
- частоту выпадение игрушки(веса в %)
8) Организовать массивы для хранения полей каждой строки
9) Используя API коллекцию: java.util.PriorityQueue добавить элементы в коллекцию
10) Организовать общую очередь: создать методы для добавления элементов (Put) и считывания (Get)
11) Вызвать метод Get 10 раз и записать результат в файл

Программа представляет из себя описание автомата типа барабана (class ToysMachine), куда загружают игрушки.
Перед игрой представляется, что игрушки уже загружены. При запуске приложения (class Main), создается файл в который 
записываются разыгрываемые призы. В консоли идет диолог с игроком. Если игрок выигрывает, случайным образом выпадает приз. 
В файле Linux ответы к заданиям 1-5.
